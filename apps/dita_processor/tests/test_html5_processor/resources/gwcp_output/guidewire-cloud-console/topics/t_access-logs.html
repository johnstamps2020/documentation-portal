Accessing application logs

You can access logs for deployed applications.

  1. From the left-hand navigation, select a project.
  2. Click Environments.
  3. Click Application Logs.
    You will be redirected to Sumo Logic.
    If you do not see the Application Logs link, contact your customer partner.
