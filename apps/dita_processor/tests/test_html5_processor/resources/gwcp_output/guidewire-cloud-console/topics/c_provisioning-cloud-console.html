Guidewire Cloud Console

Guidewire Cloud Console (GCC) is a one-stop control center for Guidewire Cloud.

As a developer, you can use Cloud Console to:
  • Access your source code and build pipelines
  • Deploy and undeploy Guidewire applications
  • Access the applications you have deployed
  • Access the application logs
As an administrator, you can use Cloud Console to:
  • Create and delete environments
  • Grant access to Guidewire Cloud
