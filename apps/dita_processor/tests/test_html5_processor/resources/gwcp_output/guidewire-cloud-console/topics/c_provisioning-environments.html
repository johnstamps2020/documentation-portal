Provisioning environments

If you need additional application environments, initiate a request with your customer partner.

If you need additional application environments, initiate a request with Guidewire Cloud Operations.

You can add new environments using Guidewire Cloud Console.

Parent topic: Cloud Console Guide

Add an environment

  1. From the left-hand navigation, select a project.
  2. Click Environments.
  3. Click New Environment.
  4. Provide a name for the new environment.
    The name must consist only of lower-case letters and digits. The name must be 253 characters or less.
  5. Optionally, select integrations.
    For the Cloud Data Access integration, select CDA enabled. For the DataHub / InfoCenter integration, select DH/IC enabled.
    You cannot clear or delete environments with integrations. When redeploying an application on an integrated environment, you cannot drop the database.
  6. Click Save.

Delete an environment

You can delete an environment when it is no longer needed. You cannot delete an environment integrated with Cloud Data Access or DataHub / InfoCenter.

  1. From the left-hand navigation, select a project.
  2. Click Environments.
  3. Expand the environment you want to delete.
  4. Click Delete and confirm.
