View logs in real time

When viewing logs in real time, you cannot use the Kubernetes fields for querying. Use the following fields:

  • _sourceCategory – Displays logs for the specified deployment.
  • _sourceName – Displays logs for the specified pod and containter.

For more details about this feature, refer to Sumo Logic documentation: Live Tail.

  1. In Sumo Logic, click New > Live Tail.
  2. Enter the query.
