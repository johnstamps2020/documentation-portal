Add a secret

  1. From the left-hand navigation, select a project.
  2. Click Secret Management.
  3. Specify the range of environments or applications to which the secret applies. Select the proper values from the Environment and Application lists.
  4. In Key, provide the identifier for the secret.
    Use only lower case letters and periods. Use periods only between letters. Do not use two or more periods in a row. For example: some.key
  5. In Value, provide the data to store. For authentication data, use the format: user:password. For example: user1:Password!123
  6. Click Add Secret.
  7. Deploy the related applications.
