Accessing source code

You can access the application source code from Guidewire Cloud Console.

  1. From the left-hand navigation, select a project.
  2. Click Environments.
  3. Click the Source Code link.
    You will be redirected to Bitbucket.
  4. Find the project you want to work on.
  5. Clone the repository using the HTTPS format and a personal access token.
    You can create personal access tokens in the Account screen in Bitbucket.
