Developing custom applications

You can develop your own custom applications that access InsuranceSuite resources and data. To make calls to InsuranceSuite applications, use the InsuranceSuite System REST APIs.

InsuranceSuite applications are consumer applications. By using the system APIs, your application can make calls into InsuranceSuite applications. These calls use the Guidewire Hub for auth functionality.
