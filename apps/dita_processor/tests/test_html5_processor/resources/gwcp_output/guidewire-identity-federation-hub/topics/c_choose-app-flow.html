Choosing an OAuth 2.0 application flow

Guidewire Hub supports the following OAuth 2.0 flows for a browser-based application:
Authorization code flow
For clients that have both a JavaScript client side and a trusted server side.
Authorization code flow with PKCE
For clients that have a JavaScript client side but no trusted server side.
