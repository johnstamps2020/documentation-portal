Accessing Bitbucket

About this task

You access Bitbucket from Guidewire Cloud Console.

Procedure

  1. In the Environments tab, click the Source Code link.
  2. Find the project you want to work on. You can find the name of the project in Cloud Console, in the drop-down at the top of the application screen.
