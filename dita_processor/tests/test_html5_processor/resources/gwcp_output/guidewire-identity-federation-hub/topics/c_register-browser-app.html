Register your browser-based application

When your organization wants to add a new application to access Guidewire data, you must go through the application registration process. Contact Guidewire Cloud Services and provide the following information:
  • The name of your application. Choose a name in the form of companyName_appName.
  • The names of the groups you will use that represent roles. See Specify roles for browser-based applications
  • The data access level that your application supports. See Choose a data access level.
  • The form that your SAML assertion will take for all the required claims. Guidewire will then configure the appropriate mappings on Guidewire Hub.
Guidewire registers your application and provides you with the following:
authServerUri
The URI for Guidewire Hub.
clientId
A unique identifier for the application that the client uses in any requests to the Guidewire Hub. The clientId is not secret, and need not be stored securely.
clientSecret
A secret string shared between the client and Guidewire Hub. The client secret must be stored securely.
