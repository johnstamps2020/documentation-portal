Deploying builds

You can manually deploy a build.

  1. From the left-hand navigation, select a project.
  2. Click Environments.
  3. Expand the environment in which you want to deploy the build.
  4. Click New Deployment.
  5. For each application, select the branch and the build you want to deploy.
  6. Optionally, provide environment variables.
  7. Optionally, select Drop Database.
    You cannot drop the application database in an environment integrated with Cloud Data Access or DataHub / InfoCenter.
  8. Click Deploy.
