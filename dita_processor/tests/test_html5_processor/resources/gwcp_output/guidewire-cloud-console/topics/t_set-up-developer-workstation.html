Set up the developer workstation

  1. Install Git.
  2. Install the Git Large File Storage extension.
  3. Enable long path support. 
    git config --global core.longpaths true
  4. Install the JDK.
  5. Set the %JAVA_HOME environment variable to the JDK installation directory.
  6. Generate a SSH key.
  7. Add the SSH key to the project you want to work on and clone the project.
  8. Navigate to the directory with the project.
  9. Run Studio. 
    ./gwb studio
