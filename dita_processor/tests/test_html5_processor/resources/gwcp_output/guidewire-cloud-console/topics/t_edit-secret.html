Edit a secret

You can edit the value of a secret.

  1. From the left-hand navigation, select a project.
  2. Click Secret Management.
  3. In the table at the bottom of the screen, find the key you want to edit.
  4. Hover over the key you want to edit and click the pencil icon.
  5. Enter the new value and click the tick icon.
  6. Deploy the related applications.
