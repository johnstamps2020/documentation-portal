Running tests

You can run tests in TeamCity.

Currently, the following testing frameworks are supported:
  • GUnit
  • InsuranceSuite Behavior Testing Framework (ISBTF)

Behavior Testing Framework

To run behavior tests, you must install Behavior Testing Framework. For more information, refer to documentation: InsuranceSuite documentation.

Running tests

To run tests, navigate to build management in TeamCity and trigger the appropriate test.

Server tests are included in the build pipeline and are triggered automatically every time you build an application.

Smoke tests and behavior tests are not included in the build pipeline. You must run them manually.
