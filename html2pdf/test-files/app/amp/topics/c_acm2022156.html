Claim Details

When you click a claim number on the Claims tab, you can view the details of open or closed claims. If the claim is in Draft status, clicking the claim number returns you to the step of the New Claim wizard where the draft last stopped.

Summary Tab

The Summary tab provides the details of the claim in the following fields:

Field

Description

Policy

The policy number associated with the claim.

Status

The current status of the claim. The status can be Open, Draft, or Closed.

Reported By

The name of the person who reported the claim.

Loss Location

Where the loss occurred, as provided in the Details step of the New Claim wizard.

Account

The account holders name.

Date of Loss

The date the loss occurred, as provided in the Policy Select step.

Primary Contact

The name of the person to contact for the claim.

Adjuster Section

The Adjuster section of the Summary tab includes the name and contact number of the adjuster assigned to the claim.

Adding a note to the adjuster creates a note for the adjuster in ClaimCenter. The adjuster sees your note and might contact you about it. This button appears when an adjuster is assigned to the claim.

See also

Adding a Note to a Claim

Vendors Involved Section

This section of the Summary tab lists the vendors that are assigned work on the insurable item as part of the claim resolution.

The vendors involved are listed under the following headings:

Heading

Description

Name

The name of the vendor involved.

Involvement

The type of vendor, such as Service Vendor.

Notes Tab

You use the Notes tab in the Claim Details page to view and attach notes to the claim.

The information about a note is shown under the following headings:

Heading

Description

Author

The name of the person who wrote the note.

Date/Time

The date and time the note was added to the claim.

Subject

The subject of the note, a summary of the contents of the body of the note.

Note

The note itself as entered in the Note field.

See also

Adding a Note to a Claim

Documents Tab

You can use the Documents tab to attach documents to the claim and to view and download any previously uploaded documents.

The documents attached are listed under the following headings:

Heading

Description

Name

The name of the document

Date Modified

The date the document with that name was last modified.

View/Download

An icon that enables you to either view or download the document. To view the document click the View icon, which opens the document in a new tab in the browser in which you are viewing the application. To download the document, right-click the icon in the browser window and select Save Link As from the shortcut menu.

Remove

The trash icon deletes the document from the claim.

See also

Adding a Document to a Claim

Payments Tab

The Payments tab provides information on the payments made for open or closed claims.



Services Tab

The Services tab allows you to view details about the claim-related services, such as repairs or medical treatments that are provided to you by vendors. The following details about the service are available:

Heading

Description

Status

The status of the service, either In Progress, Completed or Off-track.

Service Number

The unique identifier of the service request.

Next Action

The next action that is to be taken on the requested service.

Service

The type of the requested service, such as auto body medical care, painting and so on.

Expected Completion Date

The date when the requested service is expected to be completed. If the date has not been set, the label is not available.

Vendor

The provider of the service. You can see vendorâ€™s details when you click the vendorâ€™s name.
