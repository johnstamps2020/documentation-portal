```ts
export function readJsonFile(fileName: string): { [id: string]: string } {
  const translationData: TranslationFileContent = JSON.parse(
    readFileSync(`${__dirname}/i18n/${fileName}`, { encoding: "utf-8" })
  );

  return Object.entries(translationData).reduce(
    (accumulator, [key, value]) => ({
      ...accumulator,
      [key]: value.message,
    }),
    {}
  );
}
```

In the code above, we use the `reduce()` function to convert a Docusaurus
translation file into a set of "string resources". The Docusaurus format is:

```json
{
  "docHighlighter.label": {
    "message": "toggle highlights",
    "description": "If the user opens the page from the search window, the search words are highlighted. This button turns the highlights on and off."
  },
  "anonymous.positive": {
    "message": "You are anonymous,",
    "description": "Tells the user their identity is not revealed to us"
  }
}
```

And the expected string resource is.

```json
{
  "docHighlighter.label": "toggle highlights",
  "anonymous.positive": "You are anonymous,"
}
```
