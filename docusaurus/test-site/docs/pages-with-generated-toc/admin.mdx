# Administration guide

This is the administration guide for the Project. We will cover the following
topics:

- Installation
- Configuration
- Deployment
- Maintenance

We will assume that you are using a Linux distribution, but the instructions
should be similar for other operating systems.

## Installation

### Requirements

- Python 3.6+
- PostgreSQL 9.6+
- Redis 3.2+
- Node.js 8.9+
- Yarn 1.3+

### Installation

1. Clone the repository:

   ```bash
   git clone
   ```

2. Install Python dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Install Node.js dependencies:

   ```bash
   yarn install
   ```

4. Create a PostgreSQL database:

   ```bash
   createdb -U postgres -h localhost -p 5432 -O postgres -E UTF8 -T template0 -e "myproject"
   ```

5. Create a `.env` file:

   ```bash
   cp .env.example .env
   ```

6. Edit the `.env` file and set the `DATABASE_URL` variable to the database URL:

   ```bash
   DATABASE_URL=postgres://postgres:postgres@localhost:5432/myproject
   ```

7. Run the migrations:

   ```bash
   python manage.py migrate
   ```

8. Create a superuser:

   ```bash
   python manage.py createsuperuser
   ```

9. Run the development server:

   ```bash
   python manage.py runserver
   ```

10. Open the browser to http://localhost:8000/admin/ and login with the
    superuser account.
