# Internal administration guide

This guide is for internal use only.

## How to update the documentation

1. Clone the repository
2. Run `npm install`
3. Run `npm run docs:dev`
4. Make your changes
5. Run `npm run docs:build`
6. Commit and push your changes
7. Run `npm run docs:deploy`

## How to update the website

1. Clone the repository
2. Run `npm install`
3. Run `npm run website:dev`
4. Make your changes
5. Run `npm run website:build`
6. Commit and push your changes
7. Run `npm run website:deploy`
