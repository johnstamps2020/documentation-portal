# Card

Renders a card with a title and content. Optionally, you can add an image and an
action link.

import Card from '@theme/Card';
import accountManagerImage from '@site/static/img/account-manager.png';
import Collapsible from '@theme/Collapsible';

## Card with an image and a link

<div
  style={{
    display: 'flex',
    alignItems: 'center',
    justifyContent: 'center',
    padding: '2rem',
    backgroundColor: '#f5f5f5',
    margin: '2rem 0',
  }}
>
  <Card
    title="Account Manager"
    imagePath={accountManagerImage}
    actionLink={{ label: 'See project', href: '#' }}
    cardStyles={{ maxWidth: '305px', minHeight: '442px' }}
  >
    Manage this account for me, will you?
  </Card>
</div>

<Collapsible title="Show code">

```jsx
import Card from '@theme/Card';
import accountManagerImage from '@site/static/img/account-manager.png';

<Card
  title="Account Manager"
  imagePath={accountManagerImage}
  actionLink={{ label: 'See project', href: '#' }}
  cardStyles={{ maxWidth: '305px', minHeight: '442px' }}
>
  Manage this account for me, will you?
</Card>;
```

</Collapsible>

## Props

- `title` (required): The title of the card.
- `imagePath`: The path to the image to display in the card.
- `imageWidth`: The width of the image to display in the card.
- `imageClassName`: The class name to apply to the image. This might be useful
  if you have a custom class that, for example, prevents from zooming in on the
  image when using a Docusaurus Image Zoom plugin. The `doNotZoom` class is
  applied by default, but if you set a different class for your site, you can
  add it with this prop.
- `cardStyles`: Allows you to set additional props on the card wrapper
  component, in case you want to change the width, background color, etc. It's
  actually the `sx` prop MUI Card.
- `actionLink`: An object with the following properties:
  - `label`: is displayed at the bottom of the card as a link with an arrow
    icon.
  - `href`: the link at the bottom and the image in the card are both clickable
    to take the user to the destination indicated by `href`.
