# CardList

Displays a list of cards. If you add tags to the cards, users will be able to
filter the list.

Pass a list using the `items` prop. Each element in the list is a
[card](./card.mdx) with the additional `tags` property.

```js
import accountManagerImage from '@site/static/img/account-manager.png';
import CardList from '@theme/CardList';

const cards = [
  {
    title: 'Account Manager',
    imagePath: accountManagerImage,
    tags: ['Demo', 'Personal', 'Policyholder'],
    actionLink: {
      label: 'See project',
      href: '/docs/projects/account-manager', // absolute path from your baseUrl
    },
  },
  // ...etc.
];

<CardList items={cards} />;
```

## Behavior

- If the items have tags, a row of buttons appears at the top, one button per
  unique tag.
- The user can filter the list by clicking a button.
- The user can clear the filter by clicking the button again or clicking "All"
