# Collapsible

Renders a collapsed accordion that the user can expand to see more content.

import Collapsible from '@theme/Collapsible';
import CodeBlock from '@theme/CodeBlock';

## A simple text example

```jsx
import Collapsible from '@theme/Collapsible';

<Collapsible title="Expand to see some text">

## This is a heading

There is some Markdown here.

</Collapsible>
```
