# HeroCenteredWithImage

If you like long component names, then this is the component for you! This
component is a hero component with an image. It lays out the following elements
in two columns.

import HeroCenteredWithImage from '@theme/HeroCenteredWithImage';
import accountManagerLarge from '@site/static/img/account-manager-large.png';

<HeroCenteredWithImage
  imagePath={accountManagerLarge}
  title="Account Management"
  tags={['Demo', 'Personal', 'PolicyHolder']}
  resources={[
    {
        label: 'Figma file',
        href: '#'
    },
    {
        label: 'PNG files',
        href: '#'
    },
    {
        label: 'Config and JSON files',
        href: '#'
    },
    {
        label: 'Persona file',
        href: '#'
    },
  ]}
>

## Intro

This blueprint for Guidewire CustomerEngage™ Account Management serves as a
reference for all account-related activities. It enables policyholders to review
their policy, make modifications, download policy documents, view billing
status, and submit online payments.

## Scenario

Policyholders can use Guidewire CustomerEngage™ Account Management for a variety
of account-related activities. This includes making a change to their policy,
paying their bill, and filing a claim.

</HeroCenteredWithImage>

## Props

This component has a complex layout, so all props are required, unless indicated
otherwise.

- `title` - appears on the left
- `tags`: list of tags, e.g. `['Demo', 'Personal', 'PolicyHolder']`.
- `resources`: list of links, e.g. `[{ label: 'Figma file', href: '#' }]`.
- `imagePath`: path to the image, e.g.
  `import accountManagerLarge from '@site/static/img/account-manager-large.png';`.
- `children`: the content of the right column.
- `imageClassName` (optional): The class name to apply to the image. This might
  be useful if you have a custom class that, for example, prevents from zooming
  in on the image when using a Docusaurus Image Zoom plugin. The `doNotZoom`
  class is applied by default, but if you set a different class for your site,
  you can add it with this prop.
