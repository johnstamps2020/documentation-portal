Themes

A theme is a named collection of variables and their settings that affect the visual appearance and behavior of GenericCenter. GenericCenter provides predefined themes that you can choose from, and you can also define your own themes. Once a theme is defined, GenericCenter users can select it in the GenericCenter Settings dialog. The visual appearance of GenericCenter then immediately changes to reflect that theme.

GenericCenter themes are based on styles that are defined in the base application. For each defined theme, you set variables that override the default values used by the base styles.

A theme is represented by a theme file. Theme files are located in configuration > webresources > sass > themes. In the theme file, place the variable overrides that apply to the style elements for that theme. A theme file can contain the following:
  • variables settings that override the default style variables
  • @import statements that import other files containing variable settings

The theme file is the top level in a hierarchy of files that together compose the theme. When you set style variables in the theme file, those variables override the default variables that are set lower in the style hierarchy.
