How GenericCenter uses geocode data

GenericCenter uses geocode data for actions in rules and for geographic searches. The base configuration of GenericCenter supports the following high-level geocoding features:

  • GenericCenter user assignment and searching – GenericCenter can assign claims to users based on proximity between two addresses, such as an insured’s address or a user’s address. You can also search for users by proximity on the administration user search page. This feature requires that GenericCenter has the geocoding plugin enabled and that the GenericCenter database contains geocoded addresses
  • GenericCenter catastrophe search and heat maps – GenericCenter enables you to search for claims associated with a catastrophe and displays search results in a heat map on the Catastrophe Search page.
  • ContactManager address book searches (proximity search of vendors) – GenericCenter and ContactManager enable you to search for nearby service providers in the address book based on geographic proximity. This feature requires that:
    • GenericCenter and ContactManager are installed.
    • GenericCenter and ContactManager have the GeocodingPlugin enabled.
    • ContactManager contains geocoded addresses in its database.

To support using geocoding with user assignment and searching, you must install and register a GeocodePlugin implementation for an external geocoding service in GenericCenter. To support address book searches that use geocoding, you must install and register a GeocodePlugin implementation in both GenericCenter and ContactManager.

Important: To support address book searches with geocoding, configure and install both GenericCenter and ContactManager, and then integrate the two applications. For details, see Integrating ContactManager with Guidewire core applications.

The base configuration of GenericCenter uses geocode data on location addresses to enable you to search for nearby locations and assemble them into reinsurable location groups. This geocoding feature of GenericCenter is available only if your license for GenericCenter includes use of Guidewire Reinsurance Management.
