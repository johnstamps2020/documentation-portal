Create a payment plan

About this task

New payment plans are created using the Payment Plan Wizard. These are the minimal steps to create a new payment plan. Depending on the business requirements, you may need to provide values to additional fields.

Procedure

  1. Click the Administration tab.
  2. Select Actions > New Payment Plan. This starts the New Payment Plan Wizard. If multicurrency has been enabled, you must also identify the currencies the plan supports.
  3. On the first step, specify a Name and Payment Interval for the payment plan.
  4. Click Next. This advanced the wizard to the second step.
  5. Identify when there must be a down payment for pro rata charges. Depending on your choice, other fields may appear or disappear.
    • Select Every Term if a down payment is needed for the issuance term and all renewal terms.
    • Select Only First Term if a down payment is needed for the issuance term but not any renewal terms. When this is selected, a First Term section and Subsequent Terms section become visible.
    • Select None if a down payment is not needed for either the issuance term or any renewal terms. When this is selected, the Down Payment fields disappear.
  6. Provide values for the Down Payment Amount & Placement Date and First Installment Placement Date fields as needed.
  7. If there is no down payment for either the first term or the subsequent terms, an addition Start Periodic Installments With field appears. If you want the first installment to be out of sequence from the other installments, select Second Installment. When this is selected, a set of fields for scheduling the Second Installment Placement Date appears.
  8. Provide values for scheduling the One-Time Charges.
  9. Review the Summary section in the right column to confirm that the settings from both steps are correct.
  10. Click Finish.

Results

In a production system, the payment plan must be referenced by incoming billing requests. Business analysts must coordinate with integration developers to ensure that billing requests reference payment plans appropriately.

If you choose to have a down payment for the first term only, BillingCenter automatically creates a Renewal and New Renewal payment plan override for the payment plan. These are the only types of payment plan overrides that can be created from within the New Payment Plan wizard. If you need additional payment plan overrides, you must edit the payment plan after it has been created. For more information, see Additional options for charge invoicing.

The fields used in the New Payment Plan wizard are the same as the fields on the Payment Plan screen, though they appear in a different order. For an explanation of all fields in the Payment Plan screen, see The Payment Plan screen.
