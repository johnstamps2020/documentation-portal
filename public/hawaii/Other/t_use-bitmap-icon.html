Replace or add a PNG icon file

You can replace PNG icons in the base configuration, or add your own new icon file.

About this task

PNG icons are stored in the following directory: GenericCenter/modules/configuration/webresources/img/app

Procedure

  1. Do one of the following:
    • If replacing an existing icon in the base configuration, locate the icon file to replace, and then replace it with your new file of the same file type.
    • If adding a new icon file, place your new icon into the same directory as the other icons.
  2. In Guidewire Studio, click Tools > Web Resources.
