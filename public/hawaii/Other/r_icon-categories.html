Icon categories and locations

There are several categories of icons that you can use in GenericCenter, and each one is stored in a particular location.
Category of icon Description Location of icon files
PNG files Icon files using the PNG graphics format configuration > webresources > img > app
Guidewire-provided SVG files Icon files provided by Guidewire using the SVG graphics format configuration > webresources > sass > icons > svg > gw
User-provided SVG files Icon files that you can provide using the SVG graphics format configuration > webresources > sass > icons > svg > customer
Font Awesome SVG files Icon files provided by the third-party vendor Font Awesome using the SVG graphics format configuration > webresources > sass > icons > svg > fa5
CSS class A CSS class added to the gw-icon div. n/a
