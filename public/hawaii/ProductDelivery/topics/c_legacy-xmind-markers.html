Legacy support for XMind markers

APD can interpret mind maps created for a prior implementation called the Advanced Product Delivery extension. These mind maps use a subset of the XMind markers. In APD, use of these markers is deprecated.

In mind maps that use the Guidewire markers, avoid using these default markers.

If you have legacy mind maps created for Advanced Product Delivery extension, APD can convert them. These legacy mind maps used the following XMind markers:
Marker Represents
Smile face Coverage
Laugh face Condition
Angry face Exclusion
Red star Product
Orange star Product line
Blue star Risk object
Green star Exposure
Purple star Clause category
Question symbol Risk attribute category
Right symbol Yes/no attribute
Bar Chart symbol Integer attribute
Contact symbol Party attribute
Quote symbol String attribute
Flight symbol Location attribute
Line Graph symbol Decimal attribute
Medal symbol Pricing entry
Money symbol Money attribute
Pie Chart symbol Drop-down attribute
Thermometer symbol Date-time attribute
